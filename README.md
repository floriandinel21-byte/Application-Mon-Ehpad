# Mon Ehpad — Démo WebApp (GitHub Pages)

## Comptes de démonstration
- Personnel : personnel@mon-ehpad.fr / MonEhpad2025
- Direction : direction@mon-ehpad.fr / MonEhpad2025

## Installation sur GitHub Pages
1) Uploade tous les fichiers à la racine du dépôt (index.html, style.css, app.js, manifest.json, sw.js + dossier icons/)
2) GitHub → Settings → Pages → Branch: main → Folder: /(root) → Save
3) Ouvre l'URL GitHub Pages sur iPhone (Safari) → Partager → Ajouter à l'écran d'accueil

## Notes
- Démo 100% locale (pas de serveur) : une seule “instance” de données (sur l’appareil).
- Les “notifications” utilisent l’API Notification quand disponible, sinon un toast (démo).
- Messagerie : purge automatique après 30 jours.
